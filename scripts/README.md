# Scripts – BOTS v1 Lab

Aquí irán scripts de apoyo para el laboratorio, por ejemplo:

- Automatizar ingestión de datos en Splunk.
- Convertir ficheros (pcap → csv, logs → json, etc.).
- Tareas de mantenimiento o limpieza de datos.

Todos los scripts nuevos deberían tener:
- Comentarios con uso rápido.
- Ejemplos de ejecución.
